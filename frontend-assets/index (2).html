Welcome to the Green Loop Challenge

Draw one continuous loop around the grid using logic and deduction. Each number tells you exactly how many of the surrounding edges belong to the loop. Can you solve every puzzle without crossings or dead ends?

Rules

  • Numbers indicate how many edges are used.
  • The loop must be continuous.
  • No branches or crossings allowed.
  • The loop eventually returns to itself.

Edge States

  • Empty edge = undecided
  • Green line = part of loop
  • X mark = definitely not used

About Slitherlink

Slitherlink is a classic logic puzzle invented in Japan and popularized by Nikoli. Players create a single closed loop on a rectangular grid by interpreting numerical clues. The game rewards deduction and pattern recognition rather than guessing.

This version was built entirely with:

  • HTML5 Canvas for rendering the board
  • Vanilla JavaScript for gameplay logic
  • Pure CSS for all visual styling and textures
  • No external libraries or assets

Features include:

  • Multiple built-in puzzles
  • Loop validation
  • Hints and reset support
  • Responsive green-themed interface
  • Canvas interaction for precise edge selection
